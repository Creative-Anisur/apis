Class: PixelBuffer

algo.core. PixelBuffer

new PixelBuffer()

Very simple pixel buffer data structure. Works with the algo.layout.PixelViewer class. This class is design for extremely low resolution buffers for demonstrating things like Bresenham's line drawing algorithms for example
