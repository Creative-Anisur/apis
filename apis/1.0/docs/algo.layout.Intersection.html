Class: Intersection

algo.layout. Intersection

new Intersection(arg)

the generic results of various types of intersection test. For valid intersections the points property is an array of algo.layout.Vector objects. There is also a point property that returns the first point in the points array. The status property is a string that indicates why the intersection test failed if any
